# CloudCert Pro

## Setup Instructions (Bun)

1.  **Initialize Project**
    ```bash
    bun create vite my-aws-app --template react-ts
    cd my-aws-app
    ```

2.  **Install Dependencies**
    ```bash
    bun add lucide-react clsx tailwind-merge
    ```

3.  **Run Development Server**
    ```bash
    bun run dev
    ```

4.  **Build for Production**
    ```bash
    bun run build
    ```
